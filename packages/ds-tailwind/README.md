## IntelliSense

For intelliSense in VsCode add the following to `settings.json`

```
  "tailwindCSS.experimental.classRegex": [
    ["/\\*tw\\*/ ([^;]*);", "[\"'`]([^\"'`]*).*?[\"'`]", "'([^']*)'"]
  ]
```

Then add `/*tw*/` before `variants({})`
